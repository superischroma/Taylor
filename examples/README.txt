In order to run the sample programs, navigate to the directory of the taylor executable and run: "taylor <filename>" where filename is the file's local directory
For example, if the executable is in this folder and you want to test "simple_function.ty," do: "taylor simple_function.ty"
A download for the executable will be provided in the Google Drive folder.

Examples ordered based on complexity:
 - printing.ty
 - simple_function.ty
 - constants.ty
 - transcendentals.ty
 - input.ty
 - includer.ty
 - conditions.ty
